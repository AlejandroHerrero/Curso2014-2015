Requisitos para la aplicación:

Requisito 01: Java Aplication.

Requisito 02: Después de inicializarse la aplicación genera los  ficheros RDF a partir de los ficheros CSV. Lo hace automáticamente (invisible para el usuario).  Deja los ficheros RDF en la misma carpeta que los ficheros CSV.

Requisito 03: La aplicación contiene interfaz grafica sencilla.  Al pulsar el botón "Usar Datos" se genera el grafo de la información linkada sobre varios ficheros RDF.

Requisito 04: El usuario puede rastrear estos datos de manera cómoda en la pantalla central de la aplicación.
